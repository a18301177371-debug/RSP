# Data Access

The source mural images and the complete TOMBS-Mural reference bank are not publicly released in this repository. These files include mural images samples and derived component-level ROI patches that may be subject to access and reuse restrictions.

This public package provides:

- representative component-level ROI samples;
- retrieval preview examples;
- the label taxonomy and metadata schema;
- core preprocessing, retrieval.

The included files are intended for workflow demonstration and code inspection. The complete dataset is available from the corresponding author upon reasonable request.
