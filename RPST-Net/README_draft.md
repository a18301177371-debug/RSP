# TOMBS-Mural and RSPT-Net

This repository contains a public demonstration package for the TOMBS-Mural / RSPT-Net project. It is intended to document the workflow, model implementation, metadata schema and representative examples used in the study. 

## Repository Contents

- `scripts`: core preprocessing, retrieval, structure-prior generation, RSPT-Net training/inference and evaluation scripts.
- `metadata/`: schema-level metadata samples.
- `demo_data/`: representative ROI patches, damage masks, retrieval previews and contact sheets.
- `docs/`: data-access notes and method documentation.
- `pretrained/`: placeholder for model-weight release information.

## Data release policy

The complete dataset is available from the corresponding author upon reasonable request. Additional data will be released in future updates.
